tigers微框架使用文档

 

方法 说明 返回值
att主函数 该函数作为选择器载入页面 框架本身任何方法都需要依靠他来实现 你可以利用它来进行选择 ID 类 还有标签 当然这里还有很多不方便 比如只能这样选 att('#id') att('.class') att('div') 这里面有诸多的不便 以我的水平 还只能这样 还有框架本身不能链式调用 object
css方法 正如大家见到的这个方法和jq里面的一样只不过 我支持了 多组元素取值att('div').css('left') 这个会返回一组数据 当然这个有时候可能不好用 但是 还是可以用 只不过看你能不能驾驭这"高端"的返回值 Array|Number
height方法 取出元素的height值 依然是支持多组元素 可以返回多组元素的值 设置的时候 可以填PX单位也可以不填 取值：att('div').height() 设置值：att('div').height('s') Array|Number
width方法 取出元素的width值 依然是支持多组元素 可以返回多组元素的值 设置的时候 可以填PX单位也可以不填 Array|Number
left方法 取出元素的left值 依然是支持多组元素 可以返回多组元素的值 设置的时候 可以填PX单位也可以不填 Array|Number
top方法 取出元素的top值 依然是支持多组元素 可以返回多组元素的值 设置的时候 可以填PX单位也可以不填 Array|Number
html方法 可以取出元素的HTML值 返回多组数据或者一个字符串 可以直接添加HTML标签设置内容 att('div').html('<b>test!</b>') 取值:att('div').html() Array|String
text方法 取出元素的text值 不支持HTML标签的添加设置 att('div').text('a') 取值：att('div').text() Array|String
val方法 取出元素的value值 同样可以进行设置 att('div').val('x') 取值：att('div').val() Array|String
attr方法 设置元素的属性值 这里要区分开 元素的属性值并不是left top float那些 而是 id class value 这些 可以取值 设置值：att('div').attr('id','qinjia') 取值:att('div').attr('id') Array|String
choose方法 选中第几个元素然后对这个元素进行操作 att('div').choose(4) Number|Array
newElement方法 不建议使用  
El方法 返回所选元素 因为选择器不能返回选择的元素 这是我技术上的原因 att('div').El() Array
hide方法 将所选元素隐藏 att('div').hide() undefined
show方法 将所选元素显示 att('div').show() undefined
addClass方法

为所选元素添加 class类样式 att('div').addClass('qinjia') 这是添加一个 如果添加多个则需要 att('div').addClass({

key1:class1, 这里的 key可以随便写名字

key2:class2,

key3:class3

})

 undefined
delClass方法 为所选元素删除样式 这里方法有BUG 不建议用 一是会删除全部样式 只有att(this).delClass('qinjia') 才会从指定元素中 删除样式 undefined
first方法 选中第一个所选元素 att('div').first() Array
last方法 选中最后一个元素 att('div').last() Array
less方法 选中小于索引值多少的元素 att('div').less(3) Array
more方法 选中大于索引值多少的元素 att('div').more(5) Array
even 方法 选中索引值为偶数的元素 att('div').even() 如果填写值那么 even将重新计算索引值 att('div').even(3) 标志着索引值从3开始选中 Array
odd方法 选中索引值为奇数的元素 att('div').odd() 如果填写值那么odd将重新计算索引值 att('div').odd(6) 标志着索引值从6开始选中 事实上这两个函数运用好了的话完全可以当一个函数来用 Array
animate方法

这个动画函数只支持单一元素 多元素将按照元素的第一个来算
att('#div1').animate({
left:600,

opacity:40

},callback) callback是回调函数可以不写

isObj方法 判断是不是对象 att().isObj(obj) 这里可能有人会有疑问 Arr不也是对象吗？ 这还判断什么 放心 我这里的函数能清楚的分清他是不是 Array类 Object类亦或者是Function true|false
isStr方法 判断是不是字符串 att().isStr(obj) true|false
isNum方法 判断是不是数字 att().isNum(obj) true|false
isBlo方法 判断是不是布尔值 att().isBlo(obj) true|false
isUnde方法 判断是不是undefined att().isUnde(obj) true|false
isNull方法 判断是不是null att().isNull(obj) true|false
isDate方法 判断是不是日期对象 att().isDate(obj) true|false
sum方法 为数组求和 att().sum(arr) Number
max方法 为数组取出最大值 att().max(arr) Number
min方法 为数组取出最小值 att().min(arr) Number
random方法 随即返回一个数组数据 att().random(arr) str|num|blo|obj
common方法 取出几个数组中相同的数据 att().common(arr1,arr2,arr3,arr4,arr5) str|num|blo|obj
avg方法 求出数组中的平均数 att().avg(arr) Number
finNum方法 找出数组中的数字 att().finNum(arr) Number
finStr方法 找出数组中的 字符串att().finStr(arr) String
indexOf方法 找出数组中 某个元素的下标 att().indexOf(arr,val) Number

lessNum方法 选择数组中小于多少的数据 并返回一个新数组 att().lessNum(arr,10) Array
moreNum方法 选择数组中大于多少的数据 并返回一个新数组 att().moreNum(arr,35) Array
jsonLen方法 大家都知道json是没有length的 我这里提供了一个函数 计算json的length att().jsonLen(json) Number
